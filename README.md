#  Time Series Analysis Project: Presidential Approval Ratings and Economic Indicators
The Time Series Analysis Project is designed to analyze the relationship between presidential approval ratings and various economic indicators such as inflation, unemployment rate, and real GDP growth. This project aims to provide insights into how economic factors influence presidential approval ratings, which can be useful in political science research, economic forecasting, and public policy development. The core feature of this project is to load, manipulate, and visualize time series data related to presidential approval ratings and economic indicators.

##  Key Features
- Load and display historical data on presidential approval ratings and economic indicators
- Manipulate and analyze data using pandas and numpy
- Visualize data using matplotlib
- Potential for further analysis, such as data cleaning, filtering, and statistical modeling
- Integration with other notebooks or scripts for advanced analysis and modeling

## 🛠️ Tech Stack
- **Frontend:** None
- **Backend:** Python
- **Database:** CSV files
- **Data Analysis:** pandas, numpy
- **Data Visualization:** matplotlib
- **Notebook:** Jupyter Notebook

2. Install the required dependencies using pip: `pip install pandas numpy matplotlib`

1. Clone the repository to your local machine.
2. Navigate to the project directory.

4. Open the `Time_series_project (3).ipynb` notebook and start exploring.
